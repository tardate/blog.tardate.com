--- layout: post title: Request header rewrites with Java servlet filters - now on sourceforge date: '2008-06-02T23:14:00.002+08:00' author: Paul Gallagher tags: - Java - OC4J modified_time: '2008-06-03T00:10:06.885+08:00' blogger_id: tag:blogger.com,1999:blog-6220586.post-3016692074969264851 blogger_orig_url: http://blog.tardate.com/2008/06/request-header-rewrites-with-java.html --- Some time back I posted a sample and discussion of request header rewrites with Java servlet filters, and I now finally got around to setting it up with its own sourceforge project.

RewriteRequestHeaderFilter is a Java servlet filter for request header rewrites according to regex rules specified in the servlet init parameters. It is packaged as a sample application and also jar that can be inserted into any arbitrary site.
